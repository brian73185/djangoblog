# djangoblog

Installation

1.) Make sure you have python, postgresql, git, pip and virtualenv installed.<br>
2.) Create a folder to hold the project and go into that directory.<br>
3.) Create new virtual environment: virtualenv {virtualenv_name}<br>
4.) Create new database for the blog project. <br>
5.) A new folder should be created named whatever you called the virtual environment "{virtualenv_name}".  Go into that new folder.<br>
6.) Activate the new virtual environment by running the command: source bin/activate.<br>
7.) Clone the project from github: git clone https://github.com/brian73185/djangoblog.git<br>
8.) Download django: git clone https://github.com/django/django.git <br>
9.) Install required packages: pip install -r djangoblog/requirements.txt <br>
10.) Go into the project folder: cd djangoblog <br>
11.) Run the server: python manage.py runserver
